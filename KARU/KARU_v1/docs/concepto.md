# Concepto

KARU v1 es un cyberdeck portátil enfocado en funcionar como una estación offline de información, navegación y soporte.

El objetivo no es crear un ordenador portátil convencional, sino un dispositivo especializado para acceder a herramientas útiles cuando no hay conexión a internet o cuando se necesita centralizar recursos en una máquina local.

---

## Idea principal

KARU reúne en una única interfaz varios módulos:

- mapas offline
- guías y documentación
- estacion meteorologica
- wiki offline
- asistente local
- recepción de radio
- estado del sistema

Todo debe poder ejecutarse desde la propia Raspberry Pi.

---

## Filosofía

KARU prioriza funcionalidad sobre estética.

La estética cyberdeck forma parte del proyecto, pero no debe ser el centro. Cada módulo debe tener un uso real y aportar algo al dispositivo.

Nada debe estar ahí solo por apariencia.

---

## Objetivo de la versión 1

La primera versión busca validar la base funcional del proyecto:

- que la interfaz sea usable
- que el sistema pueda funcionar sin internet
- que los archivos locales se puedan consultar
- que los mapas offline carguen correctamente
- que la estructura sea ampliable

KARU v1 es el primer prototipo funcional del sistema.
