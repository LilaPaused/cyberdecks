# Roadmap

Este roadmap resume la evolución prevista de KARU v1.

---

## Fase actual

Estado: en desarrollo.

Actualmente existe una base funcional de interfaz web con navegación entre módulos, visor PDF, mapa offline y estructura inicial para integración con Kiwix.

## Criterio de versión funcional

KARU v1 se considerará funcional cuando pueda arrancar en la Raspberry Pi y permitir desde una interfaz local:

- abrir el panel principal
- navegar por los módulos
- consultar PDFs locales
- cargar mapas offline
- acceder a una wiki offline
- mostrar información básica del sistema
